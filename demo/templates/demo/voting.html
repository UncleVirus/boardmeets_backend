{POST} Create Vote

url=> {BASE_URL}/list/

  • body
  • Headers
  • Response
Body
data expected by the end point
{
"question":"string",
"description":"string",
"expiry_date": Vote expiry datetime ISO standard time e.g 2020-09-15T08:45:56.555Z,
